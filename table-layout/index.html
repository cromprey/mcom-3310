Absolute vs Relative Links

An absolute link is a hyperlink that refers to and maps to outside paths to an actual working URL. An absolute links to another website. Ex: (href="https:ualr.edu")

A relative link is a hyperlink that maps to internal paths. It will link to a file inside the website. Ex: (href="../assets/css/style.css")
