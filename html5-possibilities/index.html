HTML 5 Possibilities

We didn't always have the internet. We started in the print era, then we moved to the electronic era, but now we are in the digital era.
With the digital era, we developed the World Wide Web (www) which helped the efficiency of communication. Instead of sharing information through print, radio or TV, we introduced HTML5 and its possibilites to the world.

Back in the day, we relied soley on print media or word of mouth to spread information, but HTML has changed that standard drastically. HTML (which stands for Hyper Text Markup Language) is a coding language that consists of basic structure for web content. It can be paired with other web coding language (css, JavScript, JQuery, etc.) to make any sort of website. HTML is now a tool that we use in our everyday lives. It has allowed us to communicate with a larger audience, while also doing it faster and cheaper in the long run.
Overall, HTML5 is super cool tool!
