HTML 5 Possibilities

We didn't always have the internet. Back in the day, we relied soley on print media and word of mouth to spread information, which was the print era. We then moved to the electronic era, and now we are in the digital era.
With the digital era, we developed the internet/World Wide Web (www) which has helped the efficiency of communication. Instead of sharing information through print, radio or TV, we introduced HTML5 and its possibilites to the world.

HTML (which stands for Hyper Text Markup Language) is a coding language that consists of basic structure for web content. It can be paired with other web coding languages, like CSS and JavaScript, to make any sort of website. HTML is now a tool that we use in our everyday lives in a drastic way. We are now allowed to share more information faster and cheaper. For example, instead of going to a library and reading a book to get knowledge, we can now simply just google something and thousands of resources will pop up. There are so many cool things, and we are just scratching the tip of the iceberg.
Overall, HTML5 is super cool tool that will continue to grow and teach us things!
